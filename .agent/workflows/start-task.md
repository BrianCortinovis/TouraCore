# Start Task

Usa questo workflow all'inizio di un nuovo task su TouraCore.

Segui questi passi:

1. leggi `README.md`, `CLAUDE.md`, `docs/architecture.md`, `docs/conventions.md` e `docs/testing.md`
2. riassumi il task in 3-6 righe
3. identifica package, route, verticale e dati impattati
4. elenca i rischi principali
5. proponi un piano breve e ordinato
6. non modificare nulla finche il piano non e chiaro

Output atteso:

- contesto letto
- aree coinvolte
- piano di implementazione
- check previsti

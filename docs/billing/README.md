# Billing — Documentazione

Cartella con la spiegazione completa del sistema di billing TouraCore.

## File

- [`billing-system.html`](./billing-system.html) — guida interattiva (apri nel browser). Calcolatore live + diagrammi + scenari.
- [`billing-modifications-report.md`](./billing-modifications-report.md) — report tecnico per dev: stato, gap aperti (P0/P1/P2/P3), action plan.

## Come aprire

```bash
open docs/billing/billing-system.html
```

Funziona offline. Nessuna build, nessun framework: HTML + CSS + JS vanilla.

## Audience

- HTML → super admin / non-developer (Brian + chiunque legge per imparare)
- MD → dev follow-up (chi chiude i gap P0–P3)

# Decisions

Usa questa cartella per decisioni architetturali che vuoi ricordare nel tempo.

Formato consigliato:

- `0001-titolo-breve.md`
- `0002-titolo-breve.md`

Template minimo:

```md
# 0001 - Titolo

## Contesto

## Decisione

## Conseguenze

## Alternative valutate
```

Usa una decisione quando cambi qualcosa che altrimenti saresti costretto a rispiegare in futuro.
